# cloudinary-svelte

A [Cloudinary](https://cloudinary.com) Svelte SDK

### Test
To Run tests:
```bash
npm test # or yarn
```

### Storybook
To serve development build:
```bash
npm run storybook
```

To build static storybook site (output folder is /docs for easly sharing on github pages):
```bash
npm run build-storybook
```
